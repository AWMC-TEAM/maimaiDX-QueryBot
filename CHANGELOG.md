# 更新日志

## v26.8.1（2026-07-15）

### 新增

- `maibindfish` / `mai绑定水鱼` 支持无参数交互式绑定。
- Bot 会提供水鱼查分器入口，并引导用户在“编辑个人资料”中生成和复制
  `Import-Token`。
- Token 格式错误时继续等待，最多允许输入 3 次；支持发送“取消”退出。
- `今日舞萌` 根据每日人品值发放 BREAK：四舍五入至十位后除以 10，
  每天只可领取一次。

### 优化

- 用户提交的水鱼 Token 会优先撤回；撤回失败时明确提醒用户手动撤回。
- 绑定成功只显示脱敏 Token，并附带可供管理员追踪的 `REF_ID`。
- 水鱼绑定提示、重试与结果引用触发消息，减少多人同时操作时的歧义。
- 连续签到恢复原奖励曲线，第 5 天以后继续线性增长，不再设置奖励上限。

### 修复

- 修复发票成功后被错误报告为 `FinishedException()` 的问题。
- 一并修复账号、地图、队列、落雪 B50 等流程中同类的成功结果误报。

> 从旧版本升级不会修改或清空已有账号、BREAK、签到、Token 或成绩数据。
