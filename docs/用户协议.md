# maimaiDX QueryBot 用户协议

版本：`2026-07-15`

1. **服务性质**：本 Bot 是非官方第三方工具，与 SEGA、水鱼查分器、落雪查分器无隶属关系。查询、上传和发票可能因外部服务变化而失败。
2. **数据使用**：为完成绑定和上传，Bot 会保存用户主动提交的二维码凭据、上传 Token、账号摘要及操作时间，并向用户选择的第三方服务发送必要数据。
3. **审计与统计**：Bot 会记录脱敏 REF_ID 处理链路、命令结果、BREAK 流水和群消息计数；不在审计链路中保存消息正文、二维码或 Token 原文。
4. **BREAK**：BREAK 仅是 Bot 内部虚拟积分，不具有法定货币价值，不承诺兑现、回购或保值。转账与抽奖均为 Bot 内部互动。
5. **使用规则**：不得滥用接口、批量刷取、绕过限制、侵害他人账号或干扰群聊。管理员可限制或封禁相关功能。
6. **风险与责任**：用户应确保提交的账号与凭据属于本人或已获授权，并自行承担使用第三方接口的风险。
7. **撤回与删除**：发送“撤回用户协议”可停止后续受限功能；发送“mai解绑”可删除舞萌绑定凭据。必要的脱敏审计、安全和结算记录可按保留周期保留。

本文件是项目内的协议内容草案。Bot 实际使用的协议链接、确认词和版本由管理
WebUI 配置；用户发送 `用户协议` 获取链接，阅读后必须原样发送网页中的确认词才会
完成同意。Bot 不会在群聊里直接公布确认词。撤回同意不会自动删除既有账号绑定，
用户应另行执行 `mai解绑`。

> 这是项目内的服务协议草案，如用于公开、商业化或大规模服务，建议根据运营地法律及实际数据流程由专业人士审核。
