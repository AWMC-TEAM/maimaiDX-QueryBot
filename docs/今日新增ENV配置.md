# 今日新增 ENV 配置项

```env
# AWMC 账号与上传
AWMC_API_MODE=team
AWMC_API_BASE_URL=http://127.0.0.1:5001
AWMC_PUBLIC_GATEWAY_TOKEN=
AWMC_API_TIMEOUT_SECONDS=120
AWMC_API_RETRY_COUNT=3
AWMC_API_RETRY_DELAY_SECONDS=1
AWMC_UPLOAD_POLL_INTERVAL_SECONDS=3
AWMC_UPLOAD_POLL_TIMEOUT_SECONDS=600
AWMC_ACCOUNT_ENABLED=true
AWMC_QRCODE_CACHE_SECONDS=0

# 管理 WebUI / 审计 / 交互
MAIMAIDX_ADMIN_WEB_ENABLED=false
MAIMAIDX_ADMIN_WEB_TOKEN=replace_with_a_long_random_secret
MAIMAIDX_ADMIN_WEB_HOST=127.0.0.1
MAIMAIDX_ADMIN_WEB_PORT=8099
MAIMAIDX_ADMIN_WEB_PATH=/maimaidx/admin
MAIMAIDX_ADMIN_WEB_PUBLIC_URL=
MAIMAIDX_AUDIT_RETENTION_DAYS=90
MAIMAIDX_MESSAGE_STATS_ENABLED=true
MAIMAIDX_COMPACT_MESSAGES=true
MAIMAIDX_USER_AGREEMENT_REQUIRED=true
MAIMAIDX_KOISHI_MIGRATION_DIR=data/migration

# SQLite / YAML / MySQL
MAIMAIDX_STORAGE_BACKEND=sqlite
MAIMAIDX_STORAGE_NAMESPACE=default
MAIMAIDX_STORAGE_YAML_PATH=data/storage/state.yaml
MAIMAIDX_STORAGE_MYSQL_HOST=
MAIMAIDX_STORAGE_MYSQL_PORT=3306
MAIMAIDX_STORAGE_MYSQL_USER=
MAIMAIDX_STORAGE_MYSQL_PASSWORD=
MAIMAIDX_STORAGE_MYSQL_DATABASE=
MAIMAIDX_STORAGE_MYSQL_CHARSET=utf8mb4
MAIMAIDX_STORAGE_MYSQL_TABLE_PREFIX=maimaidx_
MAIMAIDX_STORAGE_MYSQL_SSL=false
MAIMAIDX_STORAGE_MYSQL_KEEP_SNAPSHOTS=3
MAIMAIDX_STORAGE_SYNC_INTERVAL_SECONDS=60
MAIMAIDX_STORAGE_INCLUDE_USER_SCORES=true
MAIMAIDX_STORAGE_BOOTSTRAP_POLICY=auto
MAIMAIDX_STORAGE_ALLOW_EMPTY_REMOTE_INIT=false
MAIMAIDX_STORAGE_FAIL_FAST=true
```

MySQL 至少需要填写 `HOST`、`USER`、`DATABASE`。生产环境必须使用高强度密码。
详细迁移步骤见 [统一存储与迁移说明](统一存储与迁移说明.md)。
